                                                      • дәҶи§Ј йқўеҗ‘еҜ№иұЎ еҹәжң¬жҰӮеҝө

                                                      # 01. йқўеҗ‘еҜ№иұЎеҹәжң¬жҰӮеҝө

                                                      • жҲ‘д»¬д№ӢеүҚеӯҰд№ зҡ„зј–зЁӢж–№ејҸе°ұжҳҜ йқўеҗ‘иҝҮзЁӢ зҡ„
                                                      • йқўзӣёиҝҮзЁӢ е’Ң йқўзӣёеҜ№иұЎпјҢжҳҜдёӨз§ҚдёҚеҗҢзҡ„ зј–зЁӢж–№ејҸ
                                                      • еҜ№жҜ” йқўеҗ‘иҝҮзЁӢ зҡ„зү№зӮ№пјҢеҸҜд»ҘжӣҙеҘҪең°дәҶи§Јд»Җд№ҲжҳҜ йқўеҗ‘еҜ№иұЎ

                                                      # 1.1 иҝҮзЁӢе’ҢеҮҪж•°пјҲз§‘жҷ®пјү

                                                      • иҝҮзЁӢ жҳҜж—©жңҹзҡ„дёҖдёӘзј–зЁӢжҰӮеҝө
                                                      • иҝҮзЁӢ зұ»дјјдәҺеҮҪж•°пјҢеҸӘиғҪжү§иЎҢпјҢдҪҶжҳҜжІЎжңүиҝ”еӣһеҖј
                                                      • еҮҪж•° дёҚд»…иғҪжү§иЎҢпјҢиҝҳеҸҜд»Ҙиҝ”еӣһз»“жһң

                                                      # 1.2 йқўзӣёиҝҮзЁӢ е’Ң йқўзӣёеҜ№иұЎ еҹәжң¬жҰӮеҝө

                                                      # 1) йқўзӣёиҝҮзЁӢ вҖ”вҖ” жҖҺд№ҲеҒҡпјҹ

                                                      1. жҠҠе®ҢжҲҗжҹҗдёҖдёӘйңҖжұӮзҡ„ жүҖжңүжӯҘйӘӨ д»ҺеӨҙеҲ°е°ҫ йҖҗжӯҘе®һзҺ°
                                                      2. ж №жҚ®ејҖеҸ‘йңҖжұӮпјҢе°Ҷжҹҗдәӣ еҠҹиғҪзӢ¬з«Ӣ зҡ„д»Јз Ғ е°ҒиЈ… жҲҗдёҖдёӘеҸҲдёҖдёӘ еҮҪж•°
                                                      3. жңҖеҗҺе®ҢжҲҗзҡ„д»Јз ҒпјҢе°ұжҳҜйЎәеәҸең°и°ғз”Ё дёҚеҗҢзҡ„еҮҪж•°

                                                      зү№зӮ№

                                                      1. жіЁйҮҚ жӯҘйӘӨдёҺиҝҮзЁӢпјҢдёҚжіЁйҮҚиҒҢиҙЈеҲҶе·Ҙ
                                                      2. еҰӮжһңйңҖжұӮеӨҚжқӮпјҢд»Јз ҒдјҡеҸҳеҫ—еҫҲеӨҚжқӮ
                                                      3. ејҖеҸ‘еӨҚжқӮйЎ№зӣ®пјҢжІЎжңүеӣәе®ҡзҡ„еҘ—и·ҜпјҢејҖеҸ‘йҡҫеәҰеҫҲеӨ§пјҒ

                                                      # 2) йқўеҗ‘еҜ№иұЎ вҖ”вҖ” и°ҒжқҘеҒҡпјҹ

                                                      зӣёжҜ”иҫғеҮҪж•°пјҢйқўеҗ‘еҜ№иұЎ жҳҜ жӣҙеӨ§ зҡ„ е°ҒиЈ…пјҢж №жҚ® иҒҢиҙЈ еңЁ дёҖдёӘеҜ№иұЎдёӯ е°ҒиЈ… еӨҡдёӘж–№жі•

                                                      1. еңЁе®ҢжҲҗжҹҗдёҖдёӘйңҖжұӮеүҚпјҢйҰ–е…ҲзЎ®е®ҡ иҒҢиҙЈ вҖ”вҖ” иҰҒеҒҡзҡ„дәӢжғ…пјҲж–№жі•пјү
                                                      2. ж №жҚ® иҒҢиҙЈ зЎ®е®ҡдёҚеҗҢзҡ„ еҜ№иұЎпјҢеңЁ еҜ№иұЎ еҶ…йғЁе°ҒиЈ…дёҚеҗҢзҡ„ ж–№жі•пјҲеӨҡдёӘпјү
                                                      3. жңҖеҗҺе®ҢжҲҗзҡ„д»Јз ҒпјҢе°ұжҳҜйЎәеәҸең°и®© дёҚеҗҢзҡ„еҜ№иұЎ и°ғз”Ё дёҚеҗҢзҡ„ж–№жі•

                                                      зү№зӮ№

                                                      2. жӣҙеҠ йҖӮеҗҲеә”еҜ№еӨҚжқӮзҡ„йңҖжұӮеҸҳеҢ–пјҢжҳҜдё“й—Ёеә”еҜ№еӨҚжқӮйЎ№зӣ®ејҖеҸ‘пјҢжҸҗдҫӣзҡ„еӣәе®ҡеҘ—и·Ҝ
