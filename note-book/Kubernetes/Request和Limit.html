                                                                            # Requestsе’ҢLimits

                                                                            дёҖдёӘй—®йўҳжҳҜK8sйҮҢеңЁе®ҡд№үе®№еҷЁиө„жәҗж—¶еҖҷзҡ„Requestе’ҢLimitжңүе•ҘиҒ”зі»е’ҢеҢәеҲ«гҖӮ

                                                                            е°ұжҳҜеӯ—йқўдёҠзҡ„ж„ҸжҖқпјҢrequestйҮҢе®ҡд№үзҡ„жҳҜk8sеҝ…йЎ»иҰҒдҝқиҜҒзҡ„еҗҜеҠЁиө„жәҗпјҢlimitжҳҜе°ҶжқҘе®№еҷЁиҝҗиЎҢеҸҜиғҪдҪҝз”Ёзҡ„иө„жәҗдёҠйҷҗгҖӮ

                                                                            Kube-schedulerйҖҡиҝҮrequestзҡ„е®ҡд№үжқҘеҜ»жүҫдёҖдёӘеҸҜд»Ҙж»Ўи¶ійңҖжұӮзҡ„nodeпјҢд»ҺиҖҢеңЁnodeдёҠеҗҜеҠЁеҜ№еә”зҡ„podйҮҢжүҖз”Ёзҡ„е®№еҷЁпјӣдҪҶжҳҜе®№еҷЁиҝҗиЎҢд№ӢеҗҺеӣ дёәдёҡеҠЎзҡ„еўһй•ҝжҳҜеҸҜд»ҘдҪҝз”Ёи¶…иҝҮrequestзҡ„иө„жәҗзҡ„пјҢдҪҶжҳҜжңҖй«ҳеҸӘиғҪз”ЁеҲ°limitйҮҢе®ҡд№үзҡ„иө„жәҗпјҢдҪҶжҳҜlimitйҮҢе®ҡд№үзҡ„иө„жәҗk8sжҳҜдёҚиғҪз»ҷзЎ®дҝқжҸҗдҫӣзҡ„гҖӮ
