                                                            # з®Җд»Ӣ

                                                            sysstatжҸҗдҫӣдәҶLinuxжҖ§иғҪзӣ‘жҺ§зҡ„е·Ҙе…·йӣҶпјҢеҢ…жӢ¬sarгҖҒsadfгҖҒmpstatгҖҒiostatгҖҒpidstatзӯүпјҢиҝҷдәӣе·Ҙе…·еҸҜд»Ҙзӣ‘жҺ§зі»з»ҹжҖ§иғҪе’ҢдҪҝз”Ёжғ…еҶөгҖӮеҗ„е·Ҙе…·зҡ„дҪңз”ЁеҰӮдёӢпјҡ
                                                            iostat - жҸҗдҫӣCPUз»ҹи®ЎпјҢеӯҳеӮЁI/Oз»ҹи®ЎпјҲзЈҒзӣҳи®ҫеӨҮпјҢеҲҶеҢәеҸҠзҪ‘з»ңж–Үд»¶зі»з»ҹпјү
                                                            mpstat - жҸҗдҫӣеҚ•дёӘжҲ–з»„еҗҲCPUзӣёе…із»ҹи®Ў
                                                            pidstat - жҸҗдҫӣLinuxиҝӣзЁӢзә§еҲ«з»ҹи®ЎпјҡI/OгҖҒCPUгҖҒеҶ…еӯҳзӯү
                                                            sar - ж”¶йӣҶгҖҒжҠҘе‘ҠгҖҒдҝқеӯҳзі»з»ҹжҙ»еҠЁдҝЎжҒҜпјҡCPUгҖҒеҶ…еӯҳгҖҒзЈҒзӣҳгҖҒдёӯж–ӯгҖҒзҪ‘з»ңжҺҘеҸЈгҖҒTTYгҖҒеҶ…ж ёиЎЁзӯү
                                                            sadc - зі»з»ҹжҙ»еҠЁж•°жҚ®ж”¶йӣҶеҷЁпјҢдҪңдёәsarеҗҺз«ҜдҪҝз”Ё
                                                            sa1 - ж”¶йӣҶзі»з»ҹжҙ»еҠЁж—Ҙеёёж•°жҚ®пјҢе№¶дәҢиҝӣеҲ¶ж јејҸеӯҳеӮЁпјҢе®ғдҪңдёәsadcзҡ„е·Ҙе…·зҡ„еүҚз«ҜпјҢеҸҜд»ҘйҖҡиҝҮcronжқҘи°ғз”Ё
                                                            sa2 - з”ҹжҲҗзі»з»ҹжҜҸж—Ҙжҙ»еҠЁжҠҘе‘ҠпјҢеҗҢж ·еҸҜдҪңдёәsadcзҡ„е·Ҙе…·зҡ„еүҚз«ҜпјҢеҸҜд»ҘйҖҡиҝҮcronжқҘи°ғз”Ё
                                                            sadf - еҸҜд»Ҙд»ҘCSVгҖҒXMLж јејҸзӯүжҳҫзӨәsarж”¶йӣҶзҡ„жҖ§иғҪж•°жҚ®пјҢиҝҷж ·йқһеёёж–№дҫҝзҡ„е°Ҷзі»з»ҹж•°жҚ®еҜје…ҘеҲ°ж•°жҚ®еә“дёӯпјҢжҲ–еҜје…ҘеҲ°ExcelдёӯжқҘз”ҹжҲҗеӣҫиЎЁ
                                                            nfsiostat-sysstat: жҸҗдҫӣNFS I/Oз»ҹи®Ў
                                                            cifsiostat: жҸҗдҫӣCIFSз»ҹи®Ў
                                                            sysstatеҠҹиғҪејәеӨ§пјҢеҠҹиғҪд№ҹеңЁдёҚж–ӯзҡ„еўһејәпјҢжҜҸдёӘзүҲжң¬жҸҗдҫӣдәҶдёҚеҗҢзҡ„еҠҹиғҪпјҢз”ЁжҲ·еҸҜд»ҘеҲ°sysstatе®ҳзҪ‘дәҶи§Је·Ҙе…·жңҖе…ҲеҸ‘еұ•жғ…еҶөе’ҢиҺ·еҫ—зӣёеә”зҡ„её®еҠ©жүӢеҶҢгҖӮе®ҳзҪ‘ең°еқҖпјҡ http://sebastien.godard.pagesperso-orange.fr/open in new window
