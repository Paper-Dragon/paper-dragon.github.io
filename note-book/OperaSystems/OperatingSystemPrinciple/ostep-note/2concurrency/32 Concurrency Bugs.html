                                                                                # 32 Concurrency Bugs 并发bug

                                                                                概述:
                                                                                由于并发的复杂性和不可控性,很容易产生并发bug.并发bug主要分为死锁bug和非死锁bug.非死锁bug又主要有两类,一类是违背原子性bug,一类是违背顺序性bug.

                                                                                1. 非死锁bug
                                                                                  • 违背原子性bug

                                                                                    解决方案就是直接再临界区上下加锁.
