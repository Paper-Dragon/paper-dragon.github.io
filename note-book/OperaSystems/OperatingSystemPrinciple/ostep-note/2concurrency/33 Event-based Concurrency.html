                                                                                  жҰӮиҝ°:
                                                                                  еҹәдәҺзәҝзЁӢејҖеҸ‘зҡ„еӨҡзәҝзЁӢжңүдёӨдёӘй—®йўҳ.дёҖжҳҜжҜ”иҫғе®№жҳ“еҮәй”ҷ,жӯ»й”Ғ,йҳ»еЎһ,иө„жәҗжөӘиҙ№зӯүй—®йўҳйңҖиҰҒзү№еҲ«е°ҸеҝғеӨ„зҗҶ;дәҢжҳҜеӨҡзәҝзЁӢзҡ„и°ғеәҰжҳҜз”ұж“ҚдҪңз®ЎзҗҶзҡ„,еҸҜиғҪдёҚдјҡжҢүз…§ејҖеҸ‘иҖ…жңҹжңӣзҡ„и°ғеәҰж–№ејҸиҝҗиЎҢ,еӣ жӯӨдёҚиғҪзІҫз»ҶжҺ§еҲ¶зЁӢеәҸзҡ„иҝҗиЎҢ.еҹәдәҺдәӢд»¶е№¶еҸ‘жҳҜдёҖз§Қжӣҝд»Јж–№жЎҲ.е®ғзҡ„жҖқи·ҜжҳҜзӯүеҫ…жҹҗдәӣдәӢд»¶еҸ‘з”ҹ,з„¶еҗҺеҲӨж–ӯдәӢд»¶зұ»еһӢ,дҪҝз”Ёзӣёеә”зҡ„д»Јз ҒеӨ„зҗҶ.еҹәдәҺдәӢд»¶зҡ„е№¶еҸ‘жңүдёҖдёӘйқһеёёйҮҚиҰҒзҡ„жһ¶жһ„:eventLoop().е…ёеһӢзҡ„еҹәдәҺдәӢд»¶зҡ„жңҚеҠЎеҷЁзҡ„дјӘд»Јз ҒеҰӮдёӢ:

                                                                                  while(1) {
    events = getEvents();
    for (e in events) 
        processEvent(e);
}

                                                                                  1. еҹәдәҺдәӢд»¶зҡ„йҮҚиҰҒAPI:select() (or poll)
                                                                                    select()ApiжҳҜз”ЁдәҺжҺҘж”¶дәӢд»¶зҡ„.е®ғзҡ„е®ҡд№үзұ»дјјеҰӮдёӢ:
                                                                                    int select(int nfds,
            fd_set *restrict readfds,
            fd_set *restrict writefds,
            fd_set *restrict errorfds,
            struct timeval *restrict timeout);
                                                                                    nfdsд»ЈиЎЁз”ЁжҲ·е…іеҝғзҡ„жүҖжңүж–Үд»¶жҸҸиҝ°з¬Ұзҡ„жңҖеӨ§еҖј,readfdsжҳҜз”ЁжҲ·е…іеҝғиҜ»дәӢд»¶(жҳҜеҗҰеҸҜиҜ»)зҡ„ж–Үд»¶жҸҸиҝ°з¬ҰйӣҶеҗҲ,writefdsеҲҷжҳҜи·ҹеҶҷдәӢд»¶зӣёе…ізҡ„йӣҶеҗҲ,errorfdsжҳҜи·ҹеҮәй”ҷдәӢд»¶зӣёе…ізҡ„йӣҶеҗҲ,timeoutжҳҜеҪ“жІЎжңүдәӢд»¶еҸ‘з”ҹж—¶зӯүеҫ…зҡ„и¶…ж—¶ж—¶й—ҙ.selectиҝ”еӣһеҸ‘з”ҹдәӢд»¶зҡ„дёӘж•°,еҗҢж—¶readfds,writefds,errorfdsеҶ…йғЁе…ғзҙ жӣҝжҚўдёәеҸ‘з”ҹзӣёеә”дәӢд»¶зҡ„йӣҶеҗҲ.з„¶еҗҺеҜ№иҝҷдәӣйӣҶеҗҲеҒҡзӣёеә”зҡ„еӨ„зҗҶ.дёҖдёӘз®ҖеҚ•зҡ„дҪҝз”Ёselect()д»Јз Ғ:

                                                                                    д»Һд»Јз ҒзңӢ,жҲ‘д»¬еҸ‘зҺ°жІЎжңүзәҝзЁӢзӣёе…ізҡ„й”Ғ,жқЎд»¶еҸҳйҮҸзӯү,е®һзҺ°д№ҹжҳҜжҜ”иҫғз®ҖеҚ•жҳ“жҮӮзҡ„.еӣ дёәеҗҢж—¶еҸӘдјҡжңүдёҖдёӘдәӢд»¶иў«еӨ„зҗҶ,еӣ жӯӨд№ҹдёҚйңҖиҰҒзәҝзЁӢеҲҮжҚў,е°ұдёҚдјҡеҮәзҺ°д№ӢеүҚд»Ӣз»Қзҡ„зәҝзЁӢе®үе…Ёй—®йўҳ.
                                                                                  2. еҹәдәҺдәӢд»¶зҡ„е№¶еҸ‘й—®йўҳ
                                                                                    • йҳ»еЎһзҡ„зі»з»ҹи°ғз”Ё
                                                                                      зәҜзІ№зҡ„еҹәдәҺдәӢд»¶е№¶еҸ‘жҳҜдёҚеёҢжңӣжңүйҳ»еЎһеҸ‘з”ҹзҡ„.дёҠйқўзҡ„з®ҖеҚ•е®һзҺ°жңүдёҖдёӘжҳҺжҳҫй—®йўҳе°ұжҳҜйҳ»еЎһзҡ„зі»з»ҹи°ғз”Ё.еҫҖеҫҖж¶үеҸҠеҫҲеӨҡI/Oж“ҚдҪң,иҝҷдәӣж“ҚдҪңеҸҲдјҡеј•иө·иҝӣзЁӢйҳ»еЎһ.еҰӮжһңйҳ»еЎһеҸ‘з”ҹж„Ҹе‘ізқҖдёҚиғҪжү§иЎҢе…¶е®ғж“ҚдҪң,еҜјиҮҙиө„жәҗжөӘиҙ№.иҖҢеҹәдәҺзәҝзЁӢзҡ„е№¶еҸ‘еҲҷдёҚдјҡжңүиҝҷдёӘй—®йўҳ.еҜ№дәҺиҝҷдёӘй—®йўҳ,зҺ°еңЁзі»з»ҹжңүдёҖз§ҚејӮжӯҘIOжңәеҲ¶(Asynchronous IO) еҸҜд»Ҙи§ЈеҶіиҝҷдёӘй—®йўҳ. е®ғзҡ„жҖқи·ҜжҳҜйҖҡиҝҮдёҖз§Қж•°жҚ®з»“жһ„(aiocb)и®°еҪ•IOйңҖиҰҒеҒҡзҡ„дәӢжғ…,з”ЁжҲ·и°ғз”ЁејӮжӯҘIOеҗҺй©¬дёҠиҝ”еӣһ,ејӮжӯҘIOж №жҚ®и®°еҪ•зҡ„IOдҝЎжҒҜ,иҮӘеҠЁжү§иЎҢ,е№¶жҸҗдҫӣжҺҘеҸЈз»ҷз”ЁжҲ·и°ғз”Ёд»ҘжҹҘзңӢејӮжӯҘIOжҳҜеҗҰе·Із»Ҹе®ҢжҲҗ.
                                                                                      aiocbж•°жҚ®з»“жһ„:
                                                                                      struct aiocb {
    int aio_fildes; //File descriptor
    off_t aio_offset; //File offfset
    volatile void *aio_buf;  //Location of buffer
    size_t aio_nbytes; //Length of transfer
}
                                                                                      ејӮжӯҘиҜ»IOжҺҘеҸЈ:
                                                                                      int aio_read(struct aiocb *aiocbp);
                                                                                      жҹҘзңӢејӮжӯҘIOжҳҜеҗҰе·Із»Ҹе®ҢжҲҗжҺҘеҸЈ(macзі»з»ҹ):
                                                                                      int aio_error(const struct aiocb *aiocbp);
                                                                                      иҷҪз„¶йҖҡиҝҮжҺҘеҸЈеҸҜд»ҘжҹҘиҜўејӮжӯҘе®ҢжҲҗжғ…еҶө,дҪҶжҳҜеҪ“IOиҜ·жұӮзү№еҲ«еӨҡж—¶,и°ғз”ЁжҹҘиҜўжҺҘеҸЈиҝҳжҳҜжҜ”иҫғиҖ—иҙ№иө„жәҗзҡ„.жүҖд»ҘзҺ°д»Јзі»з»ҹдёҖиҲ¬йҮҮз”ЁдҝЎеҸ·жңәеҲ¶жқҘйҖҡзҹҘеә”з”ЁIOе·Із»Ҹе®ҢжҲҗ.еә”з”ЁжҺҘж”¶еҲ°дёӯж–ӯдҝЎеҸ·еҗҺдјҡеҒңжӯўеҪ“еүҚзҡ„д»»еҠЎ,еӨ„зҗҶдёӯж–ӯ.
                                                                                    • зҠ¶жҖҒз®ЎзҗҶ
                                                                                      еҹәдәҺдәӢд»¶зҡ„е№¶еҸ‘иҝҳжңүдёҖдёӘй—®йўҳе°ұжҳҜзҠ¶жҖҒз®ЎзҗҶй—®йўҳ.еӣ дёәеҹәдәҺдәӢд»¶е№¶еҸ‘зӣёеҪ“дәҺжҠҠеҹәдәҺзәҝзЁӢзҡ„е№¶еҸ‘зҡ„жӯҘйӘӨеҲҶдёәдёҚеҗҢзҡ„дәӢд»¶,еҜ№жҜҸзұ»дәӢд»¶иҝӣиЎҢеҮ з§ҚеӨ„зҗҶ.жҜҸдёҖз§ҚдәӢд»¶еӨ„зҗҶе®ҢеҗҺ,жҺҘдёӢжқҘйңҖиҰҒе№Ід»Җд№ҲйғҪйңҖиҰҒйҖ»иҫ‘еӨ„зҗҶ.еӣ жӯӨж•ҙдёӘеЈ°жҳҺе‘ЁжңҹжүҖйңҖиҰҒзҡ„зҠ¶жҖҒдҝЎжҒҜйғҪйңҖиҰҒз®ЎзҗҶеҘҪ.жңүзӮ№зұ»дјјдәҺйҖҒеҝ«йҖ’.еҹәдәҺзәҝзЁӢе№¶еҸ‘зұ»дјјдәҺжҜҸдёӘдәәйҖҒеҝ«йҖ’жҳҜд»ҺејҖе§Ӣең°йҖҒеҲ°зӣ®зҡ„ең°,ејҖе§Ӣең°е’Ңзӣ®зҡ„ең°иҮӘе·ұйғҪжё…жҘҡ,иҖҢеҹәдәҺдәӢд»¶е№¶еҸ‘,еҝ«йҖ’жҳҜеҲҶеҮ ж®өи·ҜзЁӢйҖҒзҡ„,жҜҸдёӘдәәйҖҒдёҖж®өи·ҜзЁӢ,еҰӮжһңејҖе§Ӣең°е’Ңзӣ®зҡ„ең°дёҚз®ЎзҗҶеҘҪзҡ„иҜқ,еҸҜиғҪйҖҒдёҚеҲ°жңҖз»Ҳзӣ®зҡ„ең°.
