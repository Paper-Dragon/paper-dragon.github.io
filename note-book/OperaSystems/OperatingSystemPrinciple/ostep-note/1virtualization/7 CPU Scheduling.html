                                                                                                  # 7 и°ғеәҰзӯ–з•Ҙ

                                                                                                  жҰӮиҝ°
                                                                                                  жҜ”иҫғеҘҪзҡ„и°ғеәҰзӯ–з•ҘжҳҜиғҪдјҳе…ҲеӨ„зҗҶеҜ№е“Қеә”ж—¶й—ҙиҰҒжұӮй«ҳзҡ„д»»еҠЎ,еҗҢж—¶дёҚи®©йңҖиҰҒй•ҝж—¶й—ҙи®Ўз®—зҡ„д»»еҠЎйҘҘйҘҝ
                                                                                                  (i.e.,й”®зӣҳжү“еӯ—е°ұеұһдәҺеҜ№е“Қеә”ж—¶й—ҙиҰҒжұӮй«ҳзҡ„д»»еҠЎ,жӢ·иҙқж–Үд»¶е°ұеұһдәҺйңҖиҰҒй•ҝж—¶й—ҙи®Ўз®—cpuеҜҶйӣҶзҡ„д»»еҠЎ)
                                                                                                  дёӢйқўеҲҶжһҗеҮ з§ҚдёҚеҗҢзҡ„и°ғеәҰзӯ–з•Ҙ,е№¶еҲҶжһҗе®ғд»¬зҡ„еҲ©ејҠ

                                                                                                  • е“Қеә”ж—¶й—ҙ д»»еҠЎеҲ°жқҘеҲ°з¬¬дёҖж¬Ўиў«и°ғеәҰжү§иЎҢзҡ„ж—¶й—ҙ
                                                                                                  1. FIFO
                                                                                                    First in First out е…Ҳиҝӣе…ҲеҮәзӯ–з•Ҙ,жңүзӮ№жҳҜз®ҖеҚ•,дҪҶд№ҹеҸӘжҳҜз®ҖеҚ•,зјәзӮ№еҫҲжҳҺжҳҫ,д»»еҠЎзҡ„е‘ЁиҪ¬ж—¶й—ҙе’Ңе“Қеә”ж—¶й—ҙйғҪеҫҲе·®,еҒҮи®ҫcpuи®Ўз®—ж—¶й—ҙй•ҝзҡ„д»»еҠЎе…ҲеҲ°,еҲҷе…¶е®ғд»»еҠЎе°ұеҸӘиғҪе№ІзӯүдәҶ,йқһеёёдёҚеҸӢеҘҪ
                                                                                                  2. SJF
                                                                                                    Shortest Job First жңҖзҹӯд»»еҠЎдјҳе…Ҳзӯ–з•Ҙ, еүҚжҸҗжҳҜзҹҘйҒ“д»»еҠЎзҡ„жү§иЎҢж—¶й—ҙ,з„¶еҗҺд»»еҠЎеҗҢж—¶еҲ°жқҘзҡ„ж—¶еҖҷ,дјҳе…Ҳжү§иЎҢйңҖиҰҒж—¶й—ҙзҹӯзҡ„д»»еҠЎ зјәзӮ№:е…ҲдёҚиҜҙзҺ°е®һжғ…еҶөдёӢ,еҹәжң¬дёҚеҸҜиғҪдёҖејҖе§Ӣе°ұзҹҘйҒ“д»»еҠЎзҡ„жү§иЎҢж—¶й—ҙ. еҪ“еӨ§д»»еҠЎе…ҲеҲ°ж—¶,д№ҹдёҚеҸҜйҒҝе…Қе…¶е®ғд»»еҠЎзӯүеҫ…
                                                                                                  3. STCF
                                                                                                    Shortest Time-to-Completion First жңҖзҹӯе®ҢжҲҗж—¶й—ҙдјҳе…Ҳзӯ–з•Ҙ,еүҚжҸҗжҳҜзҹҘйҒ“д»»еҠЎзҡ„жү§иЎҢж—¶й—ҙй•ҝзҹӯ,и®©д»»еҠЎзҡ„жү§иЎҢж—¶й—ҙжҜ”еҪ“еүҚд»»еҠЎзҹӯ,еҲҷеҲҮжҚўеҲ°иҜҘд»»еҠЎжү§иЎҢ,иҝҷз§Қж–№жі•дјҳеҢ–дәҶд»»еҠЎзҡ„е№іеқҮе‘ЁиҪ¬ж—¶й—ҙ,зјәзӮ№жҳҜе®№жҳ“еј•иө·д»»еҠЎйҘҘйҘҝ,еҒҮи®ҫдёҖзӣҙйғҪжңүзҹӯд»»еҠЎеҲ°жқҘ,еҲҷй•ҝд»»еҠЎе°ұдјҡеҫ—дёҚеҲ°жү§иЎҢ,еј•еҸ‘йҘҘйҘҝ
                                                                                                  4. RR
                                                                                                    Round Robin иҪ®иҪ¬зӯ–з•Ҙ,дёҚз®Ўд»Җд№Ҳд»»еҠЎ,йғҪжҜҸйҡ”дёҖж®өж—¶й—ҙиҪ®жөҒжү§иЎҢ,еҸҜдјҳеҢ–д»»еҠЎзҡ„е“Қеә”ж—¶й—ҙ,зјәзӮ№еҫҲжҳҺжҳҫ,д»»еҠЎзҡ„е№іеқҮе‘ЁиҪ¬ж—¶й—ҙеҫҲзіҹзі•,еҗҢж—¶иҝӣзЁӢзҡ„еҲҮжҚўжҜ”иҫғйў‘з№Ғ,еҲҮжҚўзҡ„ж¶ҲиҖ—д№ҹжҜ”иҫғеӨ§
