                                                                                                    # 15 ең°еқҖиҪ¬жҚў Address Translation

                                                                                                    жҰӮиҝ°:
                                                                                                    дёҚз®ЎжңүеӨҡе°‘дёӘиҝӣзЁӢеңЁжү§иЎҢ,е®ғд»¬йғҪжҳҜе…ұдә«дёҖеӨ§еқ—4GBжҲ–жӣҙеӨҡеҶ…еӯҳ.ж“ҚдҪңзі»з»ҹе’ҢзЎ¬д»¶з»“еҗҲ,йҖҡиҝҮең°еқҖиҪ¬жҚўжҠҖжңҜ,дёҖиө·й«ҳж•Ҳ,зҒөжҙ»,е№¶жңүж•ҲжҺ§еҲ¶зқҖеҶ…еӯҳеҲҶй…Қз®ЎзҗҶ.зЎ¬д»¶еұӮйқў,

                                                                                                    1. зЎ¬д»¶еұӮйқў
                                                                                                      жҜҸдёӘcpuйғҪжңүдёҖеҜ№[еҹәеқҖ(base)еҜ„еӯҳеҷЁ] + [з•Ңйҷҗ(bound)еҜ„еӯҳеҷЁ],baseеҶіе®ҡиҝӣзЁӢеҶ…еӯҳзҡ„е®һйҷ…зү©зҗҶең°еқҖиө·зӮ№,boundеҶіе®ҡиҷҡжӢҹең°еқҖзҡ„иҢғеӣҙ,еҚіиҝӣзЁӢдёҚиғҪи®ҝй—® и¶…иҝҮиҢғеӣҙзҡ„еҶ…еӯҳ.е®һйҷ…иҝҗиЎҢж—¶,cpu йҖҡиҝҮ base+иҷҡжӢҹең°еқҖ = е®һйҷ…зү©зҗҶең°еқҖ и®ҝй—®зңҹе®һең°еқҖзҡ„ж•°жҚ®,иҫҫеҲ°ең°еқҖиҪ¬еҢ–зҡ„зӣ®зҡ„. йҖҡиҝҮиҝҷз§ҚжңәеҲ¶,иҷҪз„¶иҝӣзЁӢзҡ„еҶ…еӯҳз©әй—ҙйғҪжҳҜзӢ¬з«ӢеҲҶж•ЈеңЁж•ҙдёӘ еҶ…еӯҳзҡ„еҗ„дёӘең°ж–№,дҪҶжҳҜжҜҸдёӘиҝӣзЁӢзҡ„иҷҡжӢҹең°еқҖйғҪжҳҜд»Һ0ејҖе§Ӣ,иҝҷйғЁеҲҶеҜ№иҝӣзЁӢжҳҜйҖҸжҳҺзҡ„,еҗҢж—¶жҸҗдҫӣдҝ®ж”№baseе’Ңboundзҡ„еҠҹиғҪ,еҪ“з„¶иҝҷд№ҹеұһдәҺеҶ…ж ёзү№жқғж“ҚдҪң.

                                                                                                    2. ж“ҚдҪңзі»з»ҹеұӮйқў
                                                                                                      зЎ¬д»¶жҸҗдҫӣзҡ„base+boundжңәеҲ¶,ж–№дҫҝй«ҳж•Ҳзҡ„иҪ¬еҢ–иҷҡжӢҹең°еқҖ,дҪҶжҳҜеҜ№еҶ…еӯҳзҡ„зІҫз»Ҷз®ЎзҗҶиҝҳйңҖиҰҒж“ҚдҪңзі»з»ҹд»Ӣе…Ҙ. е“ӘдәӣеҶ…еӯҳең°еқҖжҳҜеҸҜз”Ёзҡ„,е“ӘдәӣжҳҜдёҚеҸҜз”Ёзҡ„,йғҪйңҖиҰҒ ж“ҚдҪңзі»з»ҹиҝӣиЎҢиҝҪиёӘ. иҝӣзЁӢиҝҗиЎҢд№ӢеүҚ,ж“ҚдҪңзі»з»ҹйңҖиҰҒжҢҮе®ҡиҜҘиҝӣзЁӢзҡ„е®һйҷ…зү©зҗҶең°еқҖиө·зӮ№(еҚіbase)е’ҢиҢғеӣҙ(bound). ж №жҚ®еҶ…еӯҳзҡ„еҲҶй…ҚдҪҝз”Ёжғ…еҶө,дёәдәҶжҸҗй«ҳеҶ…еӯҳзҡ„ еҲ©з”ЁзҺҮ,ж“ҚдҪңзі»з»ҹиҝҳеҸҜиғҪдјҡеҠЁжҖҒзҡ„иҪ¬з§»еҶ…еӯҳ.
