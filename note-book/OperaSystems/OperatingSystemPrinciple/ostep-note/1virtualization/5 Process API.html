                                                                                                  1. fork()зі»з»ҹи°ғз”Ё fork()з”ЁдәҺеҲӣе»әиҝӣзЁӢ,еҪ“иҝӣзЁӢи°ғз”Ёfork()ж—¶,дјҡеӨҚеҲ¶еҪ“еүҚиҝӣзЁӢжүҖжңүзҺҜеўғжқЎд»¶еҸӮж•°,з”ҹжҲҗж–°зҡ„иҝӣзЁӢ.д»–д»¬жңүеҗҢж ·зҡ„зҺҜеўғеҸӮж•°,дҪҶжҳҜpid(иҝӣзЁӢid)дёҚеҗҢ,еҗҢж—¶жӢҘжңүиҮӘе·ұзҡ„еҶ…еӯҳз©әй—ҙ

                                                                                                  2. exec()зі»з»ҹи°ғз”Ё fork()и°ғз”ЁеҲӣе»әж–°зҡ„иҝӣзЁӢеҗҺ,exec()зі»з»ҹи°ғз”ЁеҠ иҪҪе…¶е®ғзЁӢеәҸ,иҰҶеҶҷд»Јз Ғе’ҢйқҷжҖҒж•°жҚ®,еҗҢж—¶е Ҷ,ж Ҳе’Ңе…¶е®ғеҶ…еӯҳз©әй—ҙйғҪдјҡйҮҚж–°еҲқе§ӢеҢ–

                                                                                                  3. wait()зі»з»ҹи°ғз”Ё зҲ¶иҝӣзЁӢйҖҡиҝҮи°ғз”Ёwait(),зӯүеҫ…е…¶еҲӣе»әзҡ„еӯҗиҝӣзЁӢжү§иЎҢе®ҢжҲҗ,з„¶еҗҺеҶҚиҝҗиЎҢиҮӘе·ұеҗҺз»ӯж“ҚдҪң

                                                                                                  4. fork()е’Ңexec()зҡ„з»„еҗҲ йҖҡиҝҮfork()е’Ңexec()з»„еҗҲеҲӣе»әж–°иҝӣзЁӢиҝҗиЎҢдёҚеҗҢзҡ„зЁӢеәҸ. йҖҡиҝҮеҲҶзҰ»зҡ„жӯҘйӘӨз»ҷдәҲдәҶзі»з»ҹеҜ№иҝӣзЁӢзҡ„еҲӣе»әж·»еҠ ж“ҚдҪңзҡ„еҸҜиғҪ. жҜ”еҰӮйҖҡиҝҮshellж“ҚдҪңж—¶,fork()иҝ”еӣһеҗҺ,йҖҡиҝҮwait()иҜҘиҝӣзЁӢid,еҸҜд»Ҙзӯүеҫ…иҜҘиҝӣзЁӢжү§иЎҢ,жү“еҚ°иҜҘиҝӣзЁӢзҡ„иҫ“еҮәеҲ°еұҸе№•. еҶҚжҜ”еҰӮеҸҜд»Ҙfork()д№ӢеҗҺеҸҜд»ҘйҖҡиҝҮе‘Ҫд»Өж”№еҸҳ(ie,wc p3.c > newFile.txt) йҮҚе®ҡеҗ‘иҫ“еҮәз»“жһңеҲ°ж–Үд»¶дёӯ.
