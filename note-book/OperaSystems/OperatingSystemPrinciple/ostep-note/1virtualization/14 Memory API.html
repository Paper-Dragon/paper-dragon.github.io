                                                                                                  1. еҶ…еӯҳзұ»еһӢ

                                                                                                    • ж ҲеҶ…еӯҳ
                                                                                                      еҶ…еӯҳзҡ„з”іиҜ·е’ҢйҮҠж”ҫз”ұзј–иҜ‘еҷЁйҡҗејҸз®ЎзҗҶ,дёҚйңҖиҰҒз”ЁжҲ·ж“ҚдҪң.еҪ“и°ғз”ЁеҮҪж•°ж—¶,дјҡеңЁж ҲдёҠејҖиҫҹз©әй—ҙеӯҳеӮЁеҸӮж•°,жң¬ең°еҸҳйҮҸ,иҝ”еӣһеҖј,еҪ“еҮҪж•°йҖҖеҮәж—¶,йҮҠж”ҫеҶ…еӯҳ.
                                                                                                    • е ҶеҶ…еӯҳ
                                                                                                      е ҶеҶ…еӯҳзҡ„з”іиҜ·е’ҢйҮҠж”ҫйңҖиҰҒз”ұзј–зЁӢиҖ…жҳҫзӨәз®ЎзҗҶ(иҷҪз„¶й«ҳзә§иҜӯиЁҖJavaдјҡеё®еҝҷе®ҢжҲҗ),ж ҲеҶ…еӯҳзҡ„дҪңз”ЁиҢғеӣҙйғҪеңЁеҮҪж•°еҶ…,еӯҳжҙ»ж—¶й—ҙйғҪжҜ”иҫғзҹӯ.е Ҷзҡ„еҶ…еӯҳдҪңз”ЁиҢғеӣҙжҜ”иҫғеӨ§,еӯҳжҙ»ж—¶й—ҙжҜ”иҫғй•ҝ.

                                                                                                  2. еҶ…еӯҳз”іиҜ·malloc()и°ғз”Ё
                                                                                                    mallocзҡ„еҮҪж•°е®ҡд№үжҳҜ: void *malloc(size_t size); mallocйҖҡиҝҮдј е…Ҙеӯ—иҠӮж•°з”іиҜ·еҶ…еӯҳ.иҝ”еӣһзұ»еһӢдёәvoidжҳҜдёәдәҶж–№дҫҝзЁӢеәҸе‘ҳж №жҚ®йңҖиҰҒиҪ¬жҚўиҝ”еӣһзұ»еһӢ,жҜ”еҰӮиҪ¬жҚўжҲҗdouble,intзӯү

                                                                                                  3. еҶ…еӯҳйҮҠж”ҫfree()и°ғз”Ё
